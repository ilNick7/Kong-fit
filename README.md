# Kong-fit
Kong Fit è una web app semplice e minimal per la gestione delle schede di allenamento e dello storico dei workout, pensata per uso personale (e piccoli gruppi di amici). L’obiettivo è separare chiaramente:  la scheda di allenamento (uguale per tutti) i pesi e i log personali (visibili solo nell’app per l’utente corrente)
